# Front Tools

Una colección de herramientas esenciales y widgets visuales para desarrolladores frontend.

## Características
- **Múltiplos de 8:** Calculadora rápida para espaciados basados en la regla de 8pt.
- **Calculadora Clamp:** Generador visual para la propiedad CSS `clamp()` perfecta para tipografía responsiva.
- **Conversor PX / REM:** Herramienta bidireccional para convertir unidades con preview de accesibilidad.
- **Generador de Sombras:** Creador visual de `box-shadow` con capas apilables.
- **Calculadora Aspect Ratio:** Cálculo instantáneo de dimensiones para imágenes y vídeos.
- **Flexbox & Grid Playground:** Entorno visual interactivo para diseñar y copiar layouts CSS complejos en tiempo real.
- **Breakpoints Info:** Referencia visual de los anchos de pantalla estándar.

## Instalación
```bash
npm install
npm run dev
```

## Construcción
```bash
npm run build
```
*(Nota: El bundle compilará a la carpeta `/dist` y generará una copia en `/docs` lista para GitHub Pages)*
